# Inventory Add Item Page - Tailwind CSS Styling

## Overview
The inventory add item page has been completely redesigned with modern Tailwind CSS styling, featuring:
- ✅ Clean, professional interface
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Dynamic form sections based on category
- ✅ Auto-calculation of selling price (125% markup)
- ✅ Modern gradient cards and smooth transitions
- ✅ Removed all references to deleted fields

## Page Features

### 1. **Dynamic Sections**
- **Medication Details**: Shown when "Pharmaceuticals" category is selected
  - Generic Name
  - Drug Class
  - Formulation

- **Consumable Specifications**: Shown for consumable-related categories
  - Material (e.g., Plastic, Glass, Latex)
  - Size/Dimensions (e.g., 5ml, 21G, Medium)
  - Sterile checkbox

###  2. **Auto-Profit Calculation**
When you enter a "Buying Price", the system automatically:
- Calculates Selling Price as 125% of buying price (25% profit)
- Displays the profit margin below the selling price
- Example: Buying at 100 → Selling at 125 (25 profit per unit)

### 3. **Modern UI Elements**
- Gradient backgrounds on header
- Rounded cards with subtle shadows
- Smooth transitions and hover effects
- Color-coded sections (blue, emerald, indigo, amber)
- Professional typography and spacing

## How to Minify Tailwind CSS

### Option 1: Using the Provided Script (Recommended)

```bash
# Navigate to project directory
cd /home/kali/Downloads/hms

# Run the minification script
./minify_tailwind.sh
```

This will:
1. Install Tailwind CSS via npm (if not installed)
2. Create tailwind.config.js
3. Scan all templates for used classes
4. Generate minified CSS at `static/css/tailwind.min.css`

### Option 2: Manual CDN (Quick Start)

The template currently uses Tailwind CDN. To use the minified version:

1. Add to your base template (`dashboard_base.html`):
```html
{% load static %}
<link rel="stylesheet" href="{% static 'css/tailwind.min.css' %}">
```

2. Remove or comment out the CDN link:
```html
<!-- <script src="https://cdn.tailwindcss.com"></script> -->
```

### Option 3: Manual Tailwind CLI

```bash
# Install dependencies
npm init -y
npm install -D tailwindcss

# Create config
npx tailwindcss init

# Create input.css
echo "@tailwind base; @tailwind components; @tailwind utilities;" > input.css

# Build minified CSS
npx tailwindcss -i ./input.css -o ./static/css/tailwind.min.css --minify --watch
```

## Key Tailwind Classes Used

### Layout & Structure
- `min-h-screen` - Full viewport height
- `max-w-5xl mx-auto` - Centered container
- `space-y-6` - Vertical spacing
- `grid grid-cols-1 md:grid-cols-2` - Responsive grid

### Components
- `rounded-3xl` - Extreme rounded corners
- `shadow-xl` - Large shadows
- `backdrop-blur-xl` - Blur effect
- `border border-slate-200/60` - Semi-transparent borders

### Colors & Gradients
- `bg-gradient-to-br from-slate-50 via-blue-50 to-indigo-50`
- `from-slate-900 to-slate-700`
- `from-indigo-500 to-purple-600`
- `from-amber-500 to-orange-600`

### Interactive States
- `hover:shadow-xl` - Shadow on hover
- `focus:border-blue-500` - Border color on focus
- `focus:ring-4 focus:ring-blue-50` - Focus ring
- `transition-all duration-200` - Smooth transitions
- `active:scale-[0.98]` - Click effect

### Typography
- `text-2xl font-bold` - Large headings
- `text-sm font-semibold` - Labels
- `tracking-tight` - Letter spacing

## Responsive Breakpoints
